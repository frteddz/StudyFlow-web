# StudyFlow

> Study planner with Pomodoro timer — organize subjects, track tasks, and stay focused.

This repository contains only the built distribution files. The source code is not publicly available.

## Features

- Create and manage subjects, courses, and study topics
- Built-in Pomodoro timer with customizable focus and break intervals
- Track study sessions with detailed history and statistics
- Add tasks with priority, due dates, and estimated duration
- Visual calendar view to plan your study schedule
- Mark sessions complete and review productivity trends
- Set daily and weekly study goals
- Receive desktop notifications for timer and reminders
- Export study logs as CSV for external analysis

## Why StudyFlow?

StudyFlow stores all your data locally in your browser — no accounts, no cloud, no tracking. Your study habits and personal schedule stay completely private. It's free, works offline, and is designed to help students and self-learners build consistent study routines without distractions or subscriptions.
